ng-bootstrap-select-alt

A bootstrap-select inspired alternative for angular

View project on GitHub

What is this?

This is a a library to generate bootstrap dropdowns (or select's) based on bootstrap-select for angular.

Why? There are many libraries like this already.

Yes, but I don't really dig the already existing ones.

Features

Single select

<div ng-init="brands = ['McDonalds', 'Burger King', 'Wendys']"> <p> Selected brands: {{ singleBrands }} </p> <bs-select bs-model="singleBrands"> <ul> <li ng-repeat="brand in brands track by brand" bs-option="brand">{{ brand }} </li> </ul> </bs-select> </div>

Multi select

<div ng-init="brands = ['McDonalds', 'Burger King', 'Wendys']"> <p> Selected brands: {{ multiBrands }} </p> <bs-select bs-model="multiBrands" bs-multiple="true"> <ul> <li ng-repeat="brand in brands track by brand" bs-option="brand">{{ brand }} </li> </ul> </bs-select> </div>

Custom labels

<script type="text/javascript-lazy"> //IM INJECTING $scope GLOBALLY FOR THIS TO WORK, DON'T DO THAT $scope.labelFn = function(model) { return model.length + ' brands selected.' } </script> <div ng-init="brands = ['McDonalds', 'Burger King', 'Wendys']"> <p> Selected brands: {{ customLabelBrands }} </p> <bs-select bs-model="customLabelBrands" bs-empty-message="(No brands selected)" bs-multiple="true" bs-selected-message-fn="labelFn"> <ul> <li ng-repeat="brand in brands track by brand" bs-option="brand">{{ brand }} </li> </ul> </bs-select> </div>

Unselectable options

<script type="text/javascript-lazy"> //IM INJECTING $scope GLOBALLY FOR THIS TO WORK, DON'T DO THAT $scope.labelFn = function(model) { return model.length + ' brands selected.' } </script> <div ng-init="brands = ['McDonalds', 'Burger King', 'Wendys']"> <p> Selected brands: {{ unselectBrands }} </p> <bs-select bs-model="unselectBrands"> <ul> <li ng-repeat="brand in brands track by brand" bs-option="brand">{{ brand }} </li> </ul> </bs-select> </div>

You can put anything in here.

<div ng-init="sfChars = ['adon', 'birdie', 'chun-li', 'dan', 'gouki', 'guy', 'ken', 'nash', 'rose', 'ryu', 'sagat', 'sodom', 'vega']"> <p> Select Street Fighter character: {{ sfChar }} </p> <bs-select bs-model="sfChar"> <ul> <li ng-repeat="char in sfChars track by char" bs-option="char"> <img ng-src="images/{{char}}.gif"> </img> {{ char }} </li> </ul> </bs-select> </div>
